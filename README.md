# 🎨 Random Color Generator

This is Day 2 of my 30 Days JavaScript Challenge.

A simple and interactive Random Color Generator built using HTML, CSS, and JavaScript.

## 🚀 Features

- Generate random HEX color on button click
- Display HEX color code
- Dynamic background color change
- Clean and responsive UI

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript (DOM Manipulation, Math.random)

## 📚 What I Learned

- How to use Math.random()
- DOM selection and manipulation
- Updating UI dynamically
- Handling click events
